# Token Sprint - AI 工具集

基于 Next.js 15 + shadcn/ui 构建的现代化 AI 工具集，提供 Token 生成速度可视化和 LLM 显存计算功能。

## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 在浏览器中打开
# http://localhost:3000
```

## 核心功能

- **Token 生成速度可视化器** - 实时体验不同的 token 生成速度对用户体验的影响
- **大模型推理显存计算器** - 计算大语言模型推理所需的 GPU 显存和数量

## 技术栈

- Next.js 15 (App Router)
- React 19 + TypeScript
- shadcn/ui + Tailwind CSS
- 支持中英文双语
